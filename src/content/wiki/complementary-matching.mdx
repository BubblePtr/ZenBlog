# Complementary Matching（互补匹配）

Polymarket 三种匹配路径中最简单的一种，也是传统交易所唯一拥有的方式。

## 机制

市场上已有流动性，买卖双方直接交换已有代币：
- Alice 以 $0.60 买入 100 个 YES
- Bob 以 $0.60 卖出 100 个 YES
- 结果：100 个 YES 从 Bob → Alice，$60 pUSD 从 Alice → Bob

## 特征

- **零和游戏**：系统总代币供应量不变
- **Gas 最低**：只涉及基础转账，不涉及 <WikiLink slug="ctf-exchange-v2" /> 的复杂 CTF 操作
- **主流路径**：成熟、流动性充足的市场中绝大多数交易
