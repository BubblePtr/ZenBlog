# Polymarket Fee Structure（手续费结构）

Polymarket 的手续费不是一个固定百分比，而是一个与概率相关的曲线。

## 公式

> Fee = C × feeRate × p × (1-p)

其中 C 是交易数量，p 是价格（概率）。

## 特征

- **p = 0.50 时手续费最高**：方差最大，不确定性最高
- **p → 0 或 1 时手续费极低**：大局已定，摩擦成本最小
- **对称性**：买入 $0.90 YES ≈ 买入 $0.10 NO（费用相等）
- **Taker 付手续费，Maker 永远免费**

## 数学之美

p(1-p) 是伯努利分布（抛硬币）的方差公式。在整个系统设计中是"上帝公式"：

1. **手续费曲线**：不确定性（方差）越高，系统收费越高
2. **信息熵体现**：在 50% 时下注，为市场提供最多新信息，代价最大

## 激励对齐

在市场最迷茫（50/50）时，早期入场者被收取最高手续费，保护 <WikiLink slug="market-making" label="做市商" /> 免受无谓冲击；在市场几乎确定时，极低手续费鼓励套利者进场把价格推向 1 或 0。

## V2 升级

从 V1 时代的 `min(p, 1-p)` 近似值升级为完美的 p × (1-p)，数学上的绝对平滑让套利者的 <WikiLink slug="polymarket-pnl-calculation" label="定价模型" /> 更精确。
