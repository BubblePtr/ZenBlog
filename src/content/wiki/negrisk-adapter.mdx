# NegRisk Adapter（负风险适配器）

Polymarket 用于处理多结果市场的智能合约，解决传统多结果市场资金效率低下的问题。

## 核心问题

4 个候选人 A/B/C/D。做空 A 在数学上 = 做多 B + 做多 C + 做多 D。分别买入需要垫付三笔钱，效率极低。

## 解决方案

`convertPositions` 函数：把 NO 合约瞬间转换成其他所有候选人的 YES + 退还现金。

**转换公式：** 退还 A × (m-1) 份现金 + A 份其他所有候选人的 YES（m = 持有的不同 NO 数量）。

## 价值守恒证明

无论哪个候选人赢，转换前后总价值严格相等。详见原始文章中的逐世界线证明。

## 单向不可逆

NO → YES + 现金可以，反向不行。

- NO 转换成 YES 时，合约把 NO 发送到黑洞地址（Burn），利用释放的抵押品空间"合成"新 YES
- 反向需要凭空创造新的抵押品，适配器没有权限动用国库资金
- 比喻：打碎鸡蛋。NO 是完整鸡蛋，转换分出蛋黄（YES）和蛋清（现金），过程守恒但不可逆

## 套利机会

如果某个候选人的 NO 价格 > 其他所有候选人 YES 价格之和：
1. 买入 NO
2. 调用 `convertPositions` 拿到现金 + 一堆 YES
3. 立刻卖掉 YES

这是多结果市场中最高阶的无风险套利策略。
